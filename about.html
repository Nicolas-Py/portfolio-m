What

This is a portfolio website. Its design is inspired by the iTerm2 light color profile and the omz update dialoge.

The main "feature" is the overview of the latest changes - and the linking to the corresponding gitHub push - on the main site

--------------------------------------------------------------------------

How

On each push a gitHub action is performed wich adds the commit history in json to the repository. On the main page some js then reads the file and populates the changes section
